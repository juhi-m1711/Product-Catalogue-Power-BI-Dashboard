# Product Catalogue Dashboard – Power BI

## Overview  
This is a **Product Catalogue Dashboard** built in **Power BI** as a practice project to improve my data analysis and data visualization skills. The dashboard helps analyze product stock, sales, pricing, and order trends using interactive visuals and filters.

---

## Objective  
- Understand stock, sales, and order performance  
- Compare products and product groups  
- Practice building interactive dashboards in Power BI  

---

## Key Insights & Features  
- KPI cards for **Total Stock, Stock Value, Order Quantity, Total Sales, and Product Price**  
- **Stock Value Trend by Month** to track changes over time  
- **Price by Product Group** using a treemap  
- **Order Quantity by Location** using a map visual  
- **Decomposition Tree** to drill down Stock Balance by **Year → Month → Day** for detailed analysis  
- Interactive slicers for **Location, Brand, Product, Product Group, and Date**  
- Multiple catalogue pages for better navigation  

---

## Tools Used  
- **Microsoft Power BI**  
- Power Query  
- DAX  

---

## Learning Outcome  
This project helped me practice creating KPIs, using DAX, working with different visuals (including the Decomposition Tree), and designing interactive dashboards focused on business insights.
